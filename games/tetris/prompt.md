# 俄罗斯方块 (Tetris)

经典俄罗斯方块游戏，使用 vanilla JavaScript + Canvas 实现。

## 游戏说明
- 使用方向键控制方块移动和旋转
- 目标是填满整行以消除得分
- 游戏随着分数增加逐渐变快

## 技术栈
- vanilla JavaScript (无框架)
- HTML5 Canvas 渲染

## 视觉风格
- 经典 4399 风格
- 绿色主题 (#6DBB3A)
- 扁平化设计，无阴影/圆角/渐变
- 方块颜色鲜艳：红、蓝、绿、黄、紫、青、橙

## 功能需求
- 开始/暂停按钮
- 分数显示
- 下一方块预览
- 游戏结束界面
- 重新开始功能
