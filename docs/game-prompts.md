## Notes

- All prompts are designed for AI image generation via Waypoint MCP
- Thumbnails should be larger (e.g., 800x600) for portal display
- Icons should be square (e.g., 256x256) for game cards
- Style: Classic 4399 nostalgia with vibrant, playful colors
- Both English and Chinese prompts provided for flexibility
